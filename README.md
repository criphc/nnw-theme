# nnw-theme
the NetNewsWire default theme, now with 
- underline instead of colored text for links
  - make link colored when mouse hovering
- no text decoration for title/header texts
- red for theme color, including link underline, qoute borders, etc.
  - perfectly fits the best macOS system accent color (i.e. red).
